# MyTasks App

MyTasks is an Android task management application built using Java in Android Studio.

## Features
- Add new tasks
- View all tasks
- Search tasks by title
- Delete tasks with confirmation dialog
- Store tasks locally using SQLite
- RecyclerView with CardView design
- Multiple activities with intent navigation

## Technologies Used
- Java
- Android Studio
- SQLite
- RecyclerView
- CardView
- DialogFragment

## How to Run
1. Open the project in Android Studio
2. Let Gradle sync finish
3. Run the app on an emulator or Android device
